new 함수는 동적으로 공간을 할당 해주는 함수이다.

형식 = new DataType
DataType으로 공간을 할당하겠다.

형식* 는  형식으로 선언된 변수 명을 뜻한다.
void* vo로 선언된것이 있다면 void* 에는 vo가 들어간다.

void* vo로 선언되려면

vo = new void가 있어야한다.

void num(void* vo)의 형식으로 동적 할당되는 함수가 있다면 
void main()//에서 쓸때는
{
num(new void)로 할당 해줘야한다.
}

함수 포인터

Callback 함수 

순수 가상함수를 재정의 할때 매개변수가 추가되어도 됨?

추상클래스 : 하나 이상의 순수 가상함수를 포함하는 클래스
클래스: 도구, 설계도 
객체: 클래스에 따라 만들어진 메모리가 할당되어있는 것
추상클래스는 메모리를 할당 받을 수 없다 즉, 객체를 만들수 없다. 인스턴스를 만들수 없다.
자식클래스를 동적할당해서 사용할 수 있다.
